# Gouvernement ouvert Canada (FR) (ouvert-canada-ca)

Gouvernement ouvert Canada (FR) is a national government open-data portal for Canada running **CKAN**, exposing a standardized catalog API over ~47,344 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/ouvert-canada-ca/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=ouvert-canada-ca-api-evangelist&utm_content=repo)
